# Heatwave

A minimal browser-based acoustic heatmap prototype built with Vite, TypeScript, and Canvas 2D.

## Overview

Heatwave is designed to simulate and visualize acoustic fields in a 2D room environment. The current prototype renders a fixed 40x20 meter room onto an 800x400 pixel HTML canvas (using a direct 20 pixels-per-meter scale).

The primary feature is a first-pass acoustic heatmap that visualizes the sound pressure level (SPL) generated by a single subwoofer placed at `(x=1m, y=1m)`.

## Current Features

*   **Minimal Stack:** Built without any UI or CSS frameworks. Just pure TypeScript and standard Canvas API.
*   **Explicit Coordinate Mapping:** Clear separation between room coordinates (meters, bottom-left origin) and canvas coordinates (pixels, top-left origin).
*   **Acoustic Simulation (V1):**
    *   Single source direct field calculation.
    *   Models distance-based attenuation using standard `-20 * log10(d)`.
    *   Calculates relative SPL in dB, normalized against the maximum value in the room.
    *   Clamps minimum distance to avoid singularities at the source location.
*   **Heatmap Rendering:**
    *   Coarse-grid simulation step for stable performance.
    *   Maps a 50dB dynamic range to an HSL color gradient: `0 dB` (peak) maps to red, and `-50 dB` (quietest) maps to blue.
    *   Renders the physical room outline and the source marker cleanly on top of the acoustic data.

## Project Structure

Current structure:

- `index.html`: app entry point
- `src/main.ts`: current rendering and simulation logic
- `src/style.css`: minimal page styling

This structure is expected to change once the solver is split into smaller modules.

## Why the project is being built this way

The project is being built in very small, inspectable steps.

Each significant change is meant to be:
1. visible in the browser
2. manually checked
3. committed separately

This keeps the math, rendering, and future acoustic changes easy to verify.

## Development

To run the development server:

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.
